## 🧠 What is BIT Library?

**BIT Library** is an open, student-built study resource hub for **Bannari Amman Institute of Technology (BIT Sathy)**. It solves a real problem: study materials are scattered across WhatsApp groups, Google Drives, and email threads — making it hard to find what you need before exams.

This platform centralizes everything:
- 📄 Lecture notes and handwritten PDFs
- 📂 Semester-wise and subject-wise organization
- 🔎 Search to instantly locate what you need
- ⚡ Lightweight — no login, no friction, works on any device

---

## ✨ Features

| Feature | Description |
|---|---|
| 📁 **Browse Resources** | Explore study materials categorized by semester and subject |
| 🏷️ **Subject-wise Filtering** | Filter content by department, semester, or subject name |
| 🔎 **Search** | Instantly search for specific topics, units, or subjects |
| 📥 **Download Materials** | Direct access to PDFs and notes — no redirects |
| 📱 **Responsive UI** | Works cleanly on mobile, tablet, and desktop |

---

## 🛠️ Tech Stack

```
Frontend   →  HTML5 + CSS3 + Vanilla JavaScript
Hosting    →  Vercel (CD via GitHub)
```

No frameworks. No build tools. Just clean, fast, dependency-free frontend code — deliberately kept minimal for accessibility and performance on low-bandwidth college networks.

## 🚀 Getting Started

### View Live
👉 [https://bit-librarary.vercel.app](https://bit-librarary.vercel.app)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/Aruneshw/bit-librarary.git

# Navigate into the project
cd bit-librarary

# Open in browser (no build step required)
open index.html
# OR use Live Server in VS Code
```

No `npm install`. No `.env` files. It just works.
